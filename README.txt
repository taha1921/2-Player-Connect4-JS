2 Player Connect4

1. Run Server.js as "node Server.js" and then open localhost:8000/client.html on your browser
2. Restarting has not been implemented. To restart the game the clients must be refreshed and if that does not work, server must be restarted
3. The game starts once two players join the game. The console will show who is player one and player two as well who's turn it is
4. The game only works for two players so far, if another player joins then it will most likey get corrupt
5. if either player refreshes during the game or leaves, the connection will be lost and game will end 
7. if all the slots get filled up the game ends in a draw
8. the files also contain a favicon file.
